Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo utilizando as tags <big> e <small>.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. Para isso usamos a tad <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que foi adicionado depois. Para isso utilizamos a tag <ins>.

Texto Sobrescrito e Subscrito

Para um texto sobrescrito utilizamos a taga ser usado quando for elevar um número X20+3. Utilizamos a tag <sup>.

Para um texto subscrito utilizamos a tag para criar a formula da H2O. Utilizamos a tag <sub>.
